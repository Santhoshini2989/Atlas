# Feature Decision Rules

Every feature should answer:

1. What user problem does this solve?
2. Does it support Atlas vision?
3. Is this the simplest solution?
4. Is it worth the development cost?

---

# MVP Success Criteria

Define how we know Atlas is working:

Examples:

- Users can manage projects.
- Teams can organize tasks.
- AI reduces planning time.
- Users find value in AI assistance.
